# Paraula Ràpida

Juego educativo de catalán para aprender vocabulario jugando. Inspirado en "Ràpids com el vent" de TV3, adaptado a formato digital PWA.

## Descripción

Paraula Ràpida es un juego de velocidad para 2 equipos. Un jugador describe palabras en catalán sin usar la palabra prohibida, su equipo adivina antes de que acabe el tiempo. Modo multijugador local + tarjetas editables.

**Objetivo:** Aprender y practicar vocabulario catalán de forma divertida.

## Características

- **Modo Partida**: Equips de 2+, ronda de 60s, sistema de passes i encerts
- **Cartes**: Base de dades inicial de 100+ paraules en català
- **Editor**: Afegeix les teves pròpies cartes per tema
- **PWA**: Instal·lable a mòbil, funciona offline
- **Disseny**: Català 100%, amb Estelada al branding

## Tecnologies

- Frontend: HTML5 + CSS3 + Vanilla JS
- Storage: LocalStorage
- PWA: Service Worker + Manifest
- Deploy: GitHub Pages
